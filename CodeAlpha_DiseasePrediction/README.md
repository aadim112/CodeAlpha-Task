## Overview
This project implements a **Disease Prediction System** using machine learning techniques.  
The system predicts the **most probable disease** based on symptoms provided by the user.

It aims to assist in **early disease detection** and awareness using data-driven analysis.

---

## Features
- Symptom-based disease prediction
- Data preprocessing and cleaning
- Machine learning model training
- Model evaluation and accuracy measurement
- Fast and reliable predictions

## Dataset
The dataset consists of:
- Multiple symptoms as input features
- Disease labels as output classes

Each row represents a patient case with associated symptoms.

## Workflow

3. **Feature Encoding**
4. **Model Training**
5. **Model Evaluation**
6. **Disease Prediction**

The project uses supervised learning algorithms such as:
- Naive Bayes
- Decision Tree
- Random Forest *(if applied)*

- Accuracy Score
